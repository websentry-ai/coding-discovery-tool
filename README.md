# Coding Discovery Tool

This repository contains scripts for discovering and extracting coding tool configurations.

## Setup

1. Create a virtual environment:
```bash
python3 -m venv venv
```

2. Activate the virtual environment:
```bash
# On macOS/Linux:
source venv/bin/activate

# On Windows:
venv\Scripts\activate
```

3. No additional dependencies are required - the scripts use only Python standard library modules.

## Usage

The scripts are located in the `scripts/` directory.

To run the AI tools discovery script:

```bash
python -m scripts.coding_discovery_tools.ai_tools_discovery
```

Or run it directly:

```bash
python scripts/coding_discovery_tools/ai_tools_discovery.py
```
